# Hospital Management System

## Description
This project is a **Hospital Management System** built using **Next.js** for the frontend and **Express.js** for the backend. 
It allows patients, doctors, and caregivers to manage appointments, medical records, and user profiles efficiently.

## Features
- User authentication (Patients, Doctors, Caregivers)
- Patient registration and profile management
- Appointment scheduling
- Medical record management
- Email notifications for registration and updates
- Responsive UI with Next.js

## Technologies Used
- Frontend: Next.js, React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT / Sessions
- Email Notifications: Nodemailer
- Version Control: Git
